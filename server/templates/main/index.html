Исследование стандарта WAI-ARIA

О проекте

Данный проект создан с целью илюстрации возможностей стандарта доступности WAI-ARIA, разрабатываемого W3C концорциумом.

Приводятся грамотные и неправильные примеры использования стандарта в различных ситуациях. Проект создан для понимания стандарта доступности, использован в качестве курсовой/дипломной работы студента Московского Государственного Психолого-педагогического Университета Попова Всеволода Сергеевича.

Вся информация, представленная на данном ресурсе взята из Официальной документации стандарта А также из других открытых источников.
